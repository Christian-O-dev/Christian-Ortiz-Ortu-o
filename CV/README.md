# CV Profesional - Práctica No Laboral

Este repositorio contiene mi currículum en formato web y un perfil profesional en Markdown para presentar mi candidatura a prácticas.

## Contenido
- `index.html`: CV visual en formato web.
- `style.css`: estilos del CV.
- `perfil-profesional.md`: biografía, habilidades técnicas, proyectos y contacto.

## Publicar en GitHub Pages
1. Sube este repositorio a GitHub.
2. Ve a `Settings` > `Pages`.
3. En `Build and deployment`, selecciona:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main` y carpeta `/root`
4. Guarda y espera 1-2 minutos.
5. Tu CV quedará en una URL similar a:  
   `https://tu-usuario.github.io/nombre-del-repo/`

## Personalización rápida
- Reemplaza todos los textos entre `[ ]` con tu información real.
- Actualiza enlaces de GitHub, LinkedIn y correo.
- Añade más proyectos en la sección de proyectos destacados.
